## ‚öô Installation

1. Clone repository

```sh
git clone https://github.com/ulysses-ck/dictapp-react
cd dictapp-react
```

2. Install dependencies

```sh
yarn install
```

3. Init app

```sh
yarn start
```

4. Build for production

```sh
yarn build
```
